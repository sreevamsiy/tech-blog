# tech-blog

Source code for my personal tech blog: [sreevamsi.dev](https://sreevamsi.dev)

The blog is built with Hugo and published as a static site.

## What I Write About

Cloud infrastructure, DevOps, automation, containers, and AI.

## Stack

- Hugo for static site generation
- GitHub Actions for CI/CD
- Amazon S3 for static hosting storage
- Amazon CloudFront for CDN
- Route 53 for DNS
- ACM for TLS certificates
- Terraform for AWS infrastructure

cd blog
